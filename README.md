## 📄 Descripción
Este repositorio contiene el *Previo del Laboratorio 4* del curso **EL3313 – Taller de Diseño Digital**.  
El documento resume los conceptos teóricos, análisis previos y preparativos necesarios para el desarrollo del laboratorio práctico.

El previo incluye:
- Contexto del laboratorio.  
- Fundamentos teóricos solicitados por el profesor.  
- Preparación conceptual para el diseño digital a implementar.

## 🧪 Contexto del laboratorio
El Laboratorio 4 forma parte del proyecto final del curso y está orientado al diseño, validación e integración de un sistema digital complejo.  
El previo sirve como base conceptual para la etapa de implementación en FPGA.
